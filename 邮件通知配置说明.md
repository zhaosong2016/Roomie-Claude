# Roomie-Claude 邮件通知配置说明

## 功能说明

系统会在以下情况发送邮件通知到 spaceone.roomie@gmail.com：

1. **新用户提交拼房信息**：包含用户基本信息和偏好
2. **匹配成功**：两个用户成功配对时
3. **新群口令生成**：有人创建新活动时

---

## 配置步骤

### 1. 获取Gmail应用专用密码

1. 登录 Gmail：spaceone.roomie@gmail.com
2. 进入 Google账号设置：https://myaccount.google.com/
3. 点击左侧"安全性"
4. 确保已开启"两步验证"（如果没开，先开启）
5. 在"两步验证"下方找到"应用专用密码"
6. 点击"应用专用密码"
7. 选择应用：邮件
8. 选择设备：其他（自定义名称），输入"Roomie-Claude"
9. 点击"生成"
10. 复制显示的16位密码（格式类似：abcd efgh ijkl mnop）

### 2. 配置服务器

SSH登录服务器：
```bash
ssh root@49.233.127.228
cd /root/Roomie-Claude
```

编辑 api.py：
```bash
nano api.py
```

找到这一段（大约在第23-32行）：
```python
# 邮件推送配置
EMAIL_CONFIG = {
    "smtp_server": "smtp.gmail.com",
    "smtp_port": 587,
    "sender_email": "spaceone.roomie@gmail.com",
    "sender_password": "",  # 需要填入Gmail应用专用密码
    "receiver_email": "spaceone.roomie@gmail.com",
    "enabled": False  # 获取密码后改为True
}
```

修改为：
```python
# 邮件推送配置
EMAIL_CONFIG = {
    "smtp_server": "smtp.gmail.com",
    "smtp_port": 587,
    "sender_email": "spaceone.roomie@gmail.com",
    "sender_password": "你的16位应用专用密码",  # 粘贴刚才复制的密码，去掉空格
    "receiver_email": "spaceone.roomie@gmail.com",
    "enabled": True  # 改为True启用邮件通知
}
```

保存并退出（Ctrl+O 保存，Ctrl+X 退出）

### 3. 重启服务

```bash
# 查找Flask进程
ps aux | grep api.py

# 杀掉旧进程（替换下面的PID为实际进程号）
kill -9 PID

# 重新启动
nohup python3 api.py > api.log 2>&1 &

# 确认启动成功
tail -f api.log
```

## 注意事项

1. 应用专用密码只显示一次，记得保存
2. 如果密码丢失，可以删除旧的重新生成
3. 邮件通知失败不会影响拼房功能正常运行
4. 如果收不到邮件，检查Gmail的垃圾邮件文件夹
