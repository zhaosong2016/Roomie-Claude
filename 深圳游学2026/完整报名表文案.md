## 导入Roomie-Claude系统的字段映射

**仅需提取以下字段用于拼房匹配：**

| 报名表字段 | Roomie系统字段 | 转换规则 |
|-----------|---------------|---------|
| 真实姓名 | name | 直接使用 |
| 微信号 | wechat_id | 直接使用 |
| 性别 | gender | 男→male, 女→female |
| （固定值） | check_in | 2026-04-09 |
| （固定值） | check_out | 2026-04-11 |
| 吸烟习惯 | smoking | 不吸烟→no, 吸烟→yes |
| 作息倾向 | schedule | 早睡鸟→early_bird, 夜猫子→night_owl |
| 抗噪等级 | noise_in | 神经衰弱→weak, 普通玩家→medium, 雷打不动→strong |
| 出声分贝 | noise_out | 寂静模式→silent, 混响模式→loud, 低音炮模式→bass |
| （固定值） | group_code | QSDSZ0410 |

**筛选条件**：
- 只有同时满足以下条件的用户才导入拼房系统：
  1. "是否需要安排住宿" = "需要安排住宿"
  2. "住宿房型" = "尽量安排双人间"
- 入住和退房日期统一为：2026-04-09 至 2026-04-11
